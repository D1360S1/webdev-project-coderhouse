¿Qué son las gemas?

En Ruby, a las librerías se las conoce como "gemas", estas al igual que cualquier librería, son paquetes que contiene código reutilizable, pueden ser utilizadas para extender la funcionalidad de una aplicación de Ruby, proporcionando una forma fácil de compartir y distribuir código entre desarrolladores.

Las gemas son ampliamente utilizadas en el desarrollo de aplicaciones de Ruby y proporcionan una forma rápida y fácil de agregar funcionalidades a una aplicación sin tener que escribir todo el código desde cero. Muchas gemas son de código abierto y pueden ser instaladas fácilmente utilizando una herramienta llamada RubyGems, que es el sistema de gestión de paquetes de Ruby.

Las gemas se utilizan en muchos proyectos de Ruby on Rails, lo que ayuda a acelerar el proceso de desarrollo al permitir que los desarrolladores utilicen código existente en lugar de tener que crear todo desde cero. Además, las gemas pueden ser actualizadas y mantenidas por la comunidad de desarrolladores de Ruby, lo que significa que el código está constantemente mejorando y evolucionando para satisfacer las necesidades de los usuarios. En resumen, las gemas son una herramienta esencial para cualquier desarrollador de Ruby y pueden ser muy útiles para mejorar la eficiencia y la calidad del código de una aplicación.

Link a RubyGems: https://rubygems.org/

Bundler

Bundler es una herramienta para administrar dependencias de Ruby en tu aplicación. Su objetivo es simplificar el proceso de mantener tus aplicaciones Ruby en ejecución al asegurarse de que todas las gemas requeridas estén instaladas y en la versión correcta.

Bundler trabaja a partir de un archivo de configuración llamado Gemfile. Este archivo contiene una lista de todas las gemas que necesita tu aplicación, así como las versiones específicas que debes usar.

Link al repositorio en Github: https://github.com/bundler/bundler

Rails

Rails es un framework de desarrollo web de código abierto escrito en Ruby. Está diseñado para hacer que la creación de aplicaciones web sea más fácil y rápida. Rails se basa en el patrón de diseño Modelo-Vista-Controlador (MVC) y proporciona una estructura predefinida para tu aplicación, lo que significa que no tienes que preocuparte por escribir una gran cantidad de código repetitivo.

Rails también viene con muchas características útiles integradas, como la administración de la base de datos y la gestión de usuarios y sesiones. Además, hay una gran cantidad de gemas disponibles para Rails que pueden agregar aún más funcionalidad a tu aplicación.

Link al repositorio en Github: https://github.com/rails/rails

Dotenv-rails

Dotenv-rails es una gema que proporciona una manera de cargar variables de entorno desde un archivo .env en tu aplicación Rails. Las variables de entorno son útiles para almacenar información confidencial, como claves de API, contraseñas de bases de datos y otras credenciales.

Con Dotenv-rails, puedes almacenar tus variables de entorno en un archivo .env en lugar de en tu código fuente. Esto significa que puedes mantener tu información confidencial fuera de tu control de versiones y reducir el riesgo de que alguien la vea.

Link al repositorio en Github: https://github.com/bkeepers/dotenv

Net-Http

La clase Net::HTTP es una gema de Ruby estándar que proporciona una manera fácil de hacer solicitudes HTTP desde tu código Ruby. Puedes usar esta gema para acceder a recursos web, como páginas web, API y servicios web.

La clase Net::HTTP proporciona una interfaz de programación de aplicaciones (API) simple pero poderosa que te permite hacer solicitudes GET, POST, PUT, DELETE y otras solicitudes HTTP. También puede manejar encabezados HTTP, cookies y autenticación.

Link al repositorio en Github: https://github.com/ruby/net-http

RestClient

RestClient es una gema popular en Ruby para enviar solicitudes HTTP y manejar respuestas. Se puede usar para interactuar con cualquier API RESTful y admite los verbos HTTP comunes, como GET, POST, PUT y DELETE. También es fácil de usar y ofrece una interfaz simple y clara.

Link al repositorio en Github: https://github.com/rest-client/rest-client

Json

JSON es una gema estándar de Ruby para manejar JSON (JavaScript Object Notation), que es un formato de intercambio de datos ligero y fácil de leer y escribir. Permite a los desarrolladores codificar y decodificar datos JSON, y proporciona un alto nivel de flexibilidad y compatibilidad.

Link al repositorio en Github: https://github.com/flori/json

Roo

Roo es una gema de Ruby para leer y escribir archivos de hojas de cálculo en diferentes formatos, incluidos Excel, OpenOffice y Google Sheets. Proporciona una interfaz simple para acceder a los datos de la hoja de cálculo y manipularlos según sea necesario. Roo es una buena opción para aquellos que necesitan trabajar con datos de hojas de cálculo en Ruby.

Link al repositorio en Github: https://github.com/roo-rb/roo
